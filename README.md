# Dodge Asteroids

A pixelated 2D asteroids game I made using [libGDX](https://libgdx.badlogicgames.com/) for a school project.

### In-game screenshot
![Imgur](https://i.imgur.com/TzdAa0w.png)

### Running the game
Windows: `gradlew desktop:run`
Linux: `./gradlew desktop:run`
